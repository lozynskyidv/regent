## üîÑ How to Regenerate Icons

If you need to update the icon design:

### 1. Edit the SVG
Edit `generate-icons-png.js` and modify the SVG template:
- Change colors (fill values)
- Adjust text size/position
- Modify font weight or letter spacing

### 2. Run the Generator
```bash
cd app-store-icons
node generate-icons-png.js
```

### 3. Copy to Production
```bash
# Copy to app
cp output/icon.png ../assets/icon.png
cp output/adaptive-icon.png ../assets/adaptive-icon.png
cp output/splash-icon.png ../assets/splash-icon.png

# Copy to website
cp output/favicon.png ../../worthview-website/public/favicon.ico
```

### 4. Commit and Deploy
```bash
# App repo
cd ../
git add assets/*.png
git commit -m "Update app icons"
git push

# Website repo
cd ../worthview-website
git add public/favicon.ico
git commit -m "Update favicon"
git push
```

### 5. Rebuild App (if needed)
```bash
# For iOS TestFlight/App Store
eas build --platform ios --profile production
```

## üì± iOS Requirements

### Required Sizes (Auto-Generated by iOS)
iOS automatically generates these from the 1024√ó1024 master:
- 180√ó180 - iPhone App Icon @3x
- 120√ó120 - iPhone App Icon @2x
- 87√ó87 - Settings @3x
- 80√ó80 - Spotlight @2x
- 60√ó60 - Spotlight @1x
- 58√ó58 - Settings @2x
- 40√ó40 - Spotlight @1x
- 29√ó29 - Settings @1x

**You only need to provide the 1024√ó1024 icon.png!**

### iOS Guidelines
- ‚úÖ Square (1:1 aspect ratio)
- ‚úÖ No transparency
- ‚úÖ No rounded corners (iOS adds them)
- ‚úÖ RGB color space
- ‚úÖ PNG format
- ‚úÖ No alpha channel

## ‚ö†Ô∏è Common Issues

### Issue: Icon not updating in app
**Cause:** Cached by Expo/iOS
**Fix:**
```bash
# Clear Expo cache
npx expo start -c

# Or rebuild
eas build --platform ios --clear-cache
```

### Issue: Favicon not showing on website
**Cause:** Browser cache
**Fix:**
- Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Clear browser cache
- Open in incognito mode

### Issue: Icon has white corners on iOS
**Cause:** Transparency in PNG
**Fix:** Ensure icon has solid background, no alpha channel
