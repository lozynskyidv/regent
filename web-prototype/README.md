# Web Prototype Archive

This folder contains the original Figma-generated web prototype code.

**Created:** January 2026  
**Purpose:** Design reference and component logic reference only  
**Tech Stack:** React + Vite + TailwindCSS + shadcn/ui

## ⚠️ Not for Production

This code is archived for reference. The production app is being built in React Native (Expo) at the project root.

## What to Reference

- **Component Logic:** Data flow, calculations, state management patterns
- **Design System:** Color variables, spacing, typography in `src/index.css`
- **User Flows:** Modal interactions, form validation
- **Mock Data:** Demo personas and sample data structures

## Do NOT Use

- Web-specific components (divs, spans)
- Tailwind class names (use React Native StyleSheet)
- localStorage (use AsyncStorage/SecureStore)
- Web routing (use Expo Router)
